# Instalasi Hostapd

## Daftar Isi
- [Persyaratan]()
- [Instalasi]()

## Persyaratan
- Sistem operasi Linux
- Akses root (`sudo`)
- Adapter Wi-Fi yang mendukung mode AP
- Koneksi internet

## Instalasi

### Debian/Ubuntu

```
sudo apt-get update
sudo apt-get install hostapd
```

### Arch Linux

```
sudo pacman -S hostapd
```

### Fedora/RHEL

```
sudo dnf install hostapd
```
