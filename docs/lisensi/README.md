## Lisensi

Hostapd didistribusikan di bawah lisensi **BSD (3-clause BSD license)**, yang juga sama dengan lisensi yang digunakan oleh `wpa_supplicant` karena keduanya dikembangkan oleh developer yang sama ([Jouni Malinen](j@w1.fi)) dan berbagi sebagian besar basis kode.

Lisensi BSD ini bersifat permisif, sehingga hostapd dapat:

- Digunakan secara bebas, baik untuk keperluan pribadi maupun komersial.
- Dimodifikasi sesuai kebutuhan.
- Didistribusikan ulang, baik dalam bentuk source code maupun binary.
- Diintegrasikan ke dalam produk closed-source, selama tetap menyertakan notice hak cipta asli dan disclaimer sesuai ketentuan lisensi.

Teks lengkap lisensi dapat ditemukan pada file `COPYING` di dalam source code hostapd, atau langsung melalui repository resminya di [w1.fi/hostapd](https://w1.fi/hostapd/) maupun mirror GitHub-nya.
